# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-17

### Added
- Production-ready package structure with `pyproject.toml`
- Comprehensive Makefile for build automation
- Professional README.md with detailed documentation
- Apache 2.0 LICENSE
- Test suite framework with pytest
- Code quality tools (black, flake8, mypy, pylint)
- Type hints and docstrings for all modules
- CI/CD configuration
- Contributing guidelines
- EditorConfig for consistent coding styles

### Changed
- Migrated from MIT to Apache 2.0 license
- Restructured project for better modularity
- Updated dependencies to latest stable versions
- Improved documentation throughout

### Fixed
- Code style inconsistencies across modules
- Missing `__init__.py` files in packages
- Dependency version conflicts

## [0.01] - 2022-05-01

### Added
- Initial release
- Data collection script for gameplay recording
- Model training with Inception V3
- Model testing and autonomous gameplay
- Support for keyboard and gamepad inputs
- Screen capture utilities
- Motion detection for stuck prevention
- Multiple neural network architectures
- Jupyter notebooks for experimentation
